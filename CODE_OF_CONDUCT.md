# Code of Conduct

## Our Pledge

We want this to be a friendly and welcoming space for everyone learning web development. Be kind, be patient, and help each other out.

## Expected Behavior

- Be respectful and inclusive
- Give constructive feedback
- Help beginners without being condescending
- Accept mistakes gracefully - we're all learning!

## Not Tolerated

- Harassment or bullying
- Discrimination of any kind
- Spam or self-promotion in issues/PRs
- Trolling or rude behavior

## Need Help?

If you experience or witness unacceptable behavior, please reach out to the project maintainer.
