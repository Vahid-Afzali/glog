# Contributing to Glog

Thanks for your interest in this project! Since this is an educational/demo project, contributions are welcome but keep it simple.

## How to Contribute

1. **Fork the repo** - Click the fork button on GitHub
2. **Clone it** - `git clone https://github.com/your-username/glog.git`
3. **Make your changes** - Play around with the CSS, add new features, fix bugs
4. **Test it** - Open `index.html` in your browser and make sure everything looks good
5. **Submit a PR** - Push your changes and create a pull request

## What Kind of Contributions?

✅ **Good ideas:**
- Improving the glass effect
- Adding color variations or themes
- Making it more responsive
- Fixing browser compatibility issues
- Adding comments to explain the CSS tricks
- Creating alternative designs

❌ **Not needed:**
- Complex JavaScript frameworks
- Backend integration
- Over-engineering simple things

## Code Style

- Keep CSS clean and commented
- Use meaningful class names
- No need for strict formatting rules - just keep it readable

## Questions?

Feel free to open an issue if you have questions or ideas!

---

*Remember: This is a learning project, so don't stress about perfection. Have fun experimenting!* 🎨
