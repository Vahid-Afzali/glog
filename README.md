

# Glog | Glassic Login Form

A simple, educational project that demonstrates how to create a glassmorphism-style login form using pure HTML and CSS.

## What is this?

GloG (Glassic Login) is a login form that uses the glassmorphism design trend - you know, those frosted-glass looking UI elements with blur effects and transparency. It's built with just HTML and CSS, no JavaScript or frameworks needed.

## Why does this exist?

This is purely for learning purposes! I created it to practice:
- CSS backdrop-filter for blur effects
- Working with transparency and rgba colors
- Creating modern UI designs without heavy frameworks
- Understanding glassmorphism design principles

## Features

- ✨ Glassmorphism design (frosted glass effect)
- 🎨 Pure HTML & CSS (no JS required)
- 💫 Backdrop blur effects
- 🔲 Transparent backgrounds with subtle shadows
- 📱 Responsive layout

## How to use

Just open the `index.html` file in your browser. That's it!

Feel free to play around with the CSS values to see how different blur amounts, opacities, and colors affect the glass effect.

## Tech Stack

- HTML5
- CSS3 (with backdrop-filter support)

## Browser Support

Note: The `backdrop-filter` property might not work in all browsers. For best results, use Chrome, Edge, or Safari.
