# Security

## This is a Static Demo Project

Glog is a purely frontend educational project consisting only of HTML and CSS files. It has:

- ❌ No backend server
- ❌ No database
- ❌ No user authentication
- ❌ No data collection
- ❌ No API keys or secrets

## What This Means

Since this is just a static login form demo:
- There are no security vulnerabilities to report
- No user data is stored or transmitted
- The "login" button doesn't actually do anything
- It's safe to host anywhere (GitHub Pages, Netlify, etc.)
